# 可视化知识图谱

目前的功能

- 初始化，显示目录关系：`Deep Learning`（书名）`-->` 各个章节

- 点击章节
  - 跳转到章首页
  - 显示前置关系：章节名 `-->` 小节名

- 点击小节
  - 跳转到小节首页
  - 显示包含关系：小节名 `-->` 实体

- 点击实体
  - 跳转到实体出现的页
  - 显示 共现、语义关系：实体 `<-->` 实体
